## 成熟策略硬门槛

成熟策略必须同时满足：

1. 收益和 Calmar 保真，不能用低收益防守线凑 PASS。
2. 训练集、测试集、验证集表现不能断裂。
3. 关键参数不能是孤岛。
4. Top20 利润占比不能超过 `100%`。
5. 真实撮合必须包含 T+1、涨跌停、停牌、手续费、滑点和 100 股整手约束。
6. 冻结后 clean-forward 至少 20 个交易日。
7. 策略记录、参数、命令和输出路径必须可复现。
